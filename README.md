# BatchVerse

This repository contains the source code, technical specifications, and project dashboard for the BatchVerse React Native app.

## Project Structure
- `batchverse-mobile/`: React Native (Expo) frontend.
- `batchverse-backend/`: Azure Functions (Node.js) backend.
- `DASHBOARD.md`: Current project status and sprint backlog.
- `BatchVerse_TechSpec.md`: Detailed technical architecture and API contract.

## Getting Started

### Backend
```bash
cd batchverse-backend
npm install
func start
```

### Frontend
```bash
cd batchverse-mobile
npm install
npx expo start
```
